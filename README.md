# Tres Tacos LLC

Website for Tres Tacos LLC, a drive-thru taco truck at 6035 Losee Rd, North Las Vegas, NV 89081.

## Development

This is a static single-page site. Open `index.html` directly in a browser, or serve it locally:

```
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
